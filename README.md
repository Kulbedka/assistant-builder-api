# Assistant Builder API

Backend API for the Assistant Builder platform.

## Tech Stack

- Node.js
- Express.js
- TypeScript
- Prisma
- PostgreSQL
- Supabase
- REST API
- PM2
- Nginx
- VPS

## Features

- Health check endpoint
- Get assistants
- Get assistant by id
- Create assistants
- Get assistant messages
- Create chat messages
- PostgreSQL database integration via Prisma

## Architecture

Client: Next.js frontend  
API: Express.js backend  
Database: PostgreSQL via Supabase  
Deployment: Ubuntu VPS with PM2 and Nginx
